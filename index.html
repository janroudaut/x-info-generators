A live demo built from real generator output — every page is a single self-contained .html with all images inlined.

Install & run

# install both commands uv tool install x-info-generators # generate a page gen-game-info /path/to/game gen-video-info /path/to/movie.mkv # generate pages across a whole library (-R = recurse) gen-video-info -R /path/to/library/ # build a catalog from the pages already generated above gen-video-info --index catalog.html /path/to/library/
